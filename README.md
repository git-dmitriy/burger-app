# Burgers-app

Practice of using react-router v6.

# Tech stack

- React.js
- [Emotion](https://emotion.sh/docs/introduction) framework agnostic
- React-router v6 beta
- JSON server
- Concurrently

![screenshot](/screenshot.png)

## Getting Started

```bash
git clone https://github.com/git-dmitriy/burger-app.git
cd burger-app
npm install
npm start

```
